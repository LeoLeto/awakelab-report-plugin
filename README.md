# Moodle Question Bank Report Plugin

## Description
This plugin allows teachers and administrators to visualize and download all questions and answers from a course's question bank. It provides an easy-to-use interface to export question data for analysis or backup purposes.

## Features
- View all questions from the course question bank
- Display question text, type, category, and answers
- Export questions and answers to CSV format
- Filter by question category
- User-friendly interface

## Installation

1. Copy the `report-plugin` folder to `{moodle-root}/report/questionbank/`
2. Log in as an administrator
3. Go to Site Administration > Notifications
4. Follow the installation prompts

## Usage

1. Navigate to a course
2. Go to Course Administration > Reports > Question Bank Report
3. View the list of questions
4. Use the "Download CSV" button to export the data

## Requirements
- Moodle 4.0 or higher
- Appropriate permissions (teacher or administrator role)
